# Image Generation Worker Professional Standard v1

## Purpose

Define professional documentary image generation standards so the Image Generation Worker reliably executes the Visual Director's evaluated plan with high fidelity and production-ready quality.

## Definition Of Professional Documentary Image Generation

Professional image generation must:
- Faithfully execute the evaluated visual plan.
- Preserve historical plausibility and contextual integrity.
- Maintain character, lighting, and style consistency across generated scene outputs.
- Produce images that are compositionally clear and video-assembly ready.
- Avoid visual artifacts or style drift that reduce documentary trust.

## Measurable Evaluation Criteria

### 1. Prompt Fidelity
- Criterion: Generated outputs match evaluated scene instructions and intent.
- Metric: Prompt Fidelity score (0-10).
- Professional threshold: >= 7.

### 2. Historical Accuracy
- Criterion: Visual outputs remain plausible for the documented setting and era cues.
- Metric: Historical Accuracy score (0-10).
- Professional threshold: >= 7.

### 3. Scene Accuracy
- Criterion: Generated scene visuals reflect requested scene structure and focus.
- Metric: Scene Accuracy score (0-10).
- Professional threshold: >= 7.

### 4. Character Consistency
- Criterion: Subject depiction is coherent across generated scene set.
- Metric: Character Consistency score (0-10).
- Professional threshold: >= 7.

### 5. Visual Quality
- Criterion: Output quality is suitable for documentary pipeline composition.
- Metric: Visual Quality score (0-10).
- Professional threshold: >= 7.

### 6. Documentary Style Consistency
- Criterion: Styling aligns with documentary tone and remains coherent across set.
- Metric: Documentary Style Consistency score (0-10).
- Professional threshold: >= 7.

### 7. Overall Image Generation Quality
- Criterion: Composite quality across all six dimensions.
- Metric: Overall score (0-10).
- Professional threshold: >= 7.

## Quality Gate Rules

- Fail if any category score < 5.
- Conditional pass if overall score is between 6.0 and 6.9 and no category < 5.
- Professional pass if overall score >= 7.0 and no category < 5.

## Required Evaluation Output

- Category scores
- Overall score
- Evidence
- Diagnosis
- Prescription
- Expected improvement
- Classification: PASS / CONDITIONAL / FAIL
