# Storytelling Worker Audience Commitment Playbook v1

## Professional Principle

Every second of a documentary must earn the audience's willingness to continue watching.

## Mental Models

### 1. Earned-Second Model
Each segment must do at least one of:
- Increase understanding
- Raise meaningful stakes
- Strengthen curiosity

### 2. Promise-Progress Model
Every narrative promise should be advanced or fulfilled within a reasonable window.

### 3. Tension-Clarity Balance
Maintain curiosity without sacrificing audience orientation.

### 4. Transition Value Model
Transitions should carry narrative value, not only structural movement.

### 5. Trust Ledger Model
Framing and pacing choices either build or erode viewer trust.

## Decision Checklist

1. Does this section create a concrete reason to keep watching?
2. Does it add new value rather than repeat prior content?
3. Is the viewer oriented enough to understand the stakes?
4. Is there an active unresolved question worth continuing for?
5. Does this section preserve factual and editorial integrity?
6. Do transitions maintain momentum without confusion?
7. If this section were removed, would commitment improve?

## Common Mistakes

- Starting with low-stakes exposition.
- Repeating the same point without progression.
- Creating artificial suspense unsupported by evidence.
- Overloading context too early.
- Abrupt tone shifts that reduce trust.
- Resolving core curiosity too early.
- Long transitions with no narrative payoff.

## Evaluation Rubric

Score each category 0-10.

- Opening Strength
- Curiosity
- Narrative Flow
- Information Density
- Audience Commitment

Rubric bands:
- 9-10: Elite professional quality
- 7-8: Professional-ready quality
- 5-6: Developing quality, revision required
- 0-4: Substandard, major rewrite required

Overall score:
- Average of five categories rounded to one decimal place.

Decision:
- PASS: overall >= 7.0 and no critical category < 5.
- CONDITIONAL: overall 6.0-6.9 and no critical category < 5.
- FAIL: overall < 6.0 or any critical category < 5.
