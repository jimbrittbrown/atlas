# Layer 7 — Worker Layer

## Purpose

Workers are the execution workforce of Atlas.

Workers perform the tasks assigned by subsystems.

Workers never receive Executive Requests directly.

Workers never make Executive decisions.

Workers specialize in execution.

---

# Responsibilities

- Execute assigned work
- Report progress
- Report completion
- Report failures
- Produce work products

# Communication

Workers communicate only with:

Subsystems

Worker Orchestration

Workers NEVER communicate directly with Executive.

Workers NEVER communicate directly with AtlasApplication.

Workers NEVER communicate directly with other departments unless coordinated.

---

# Rules

Workers execute.

Workers never govern.

Workers remain replaceable.

Workers remain independently scalable.

Workers remain stateless whenever practical.
