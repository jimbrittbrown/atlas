# Layer 6 — Subsystem Layer

## Purpose

Subsystems are Atlas departments.

Each subsystem owns a specialized business capability.

Subsystems execute domain-specific logic.

Subsystems never govern Atlas.

Subsystems never make Executive decisions.

---

# Responsibilities

- Execute specialized business logic
- Manage subsystem state
- Maintain subsystem workflows
- Produce subsystem results
- Report execution outcomes
