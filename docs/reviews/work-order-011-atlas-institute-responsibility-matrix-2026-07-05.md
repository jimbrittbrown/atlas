# Work Order 011 - Atlas Institute Responsibility Matrix

Date: 2026-07-05
Status: PLANNING ONLY

| Layer / Service | Responsibility | Out of Scope for Atlas Institute |
| --- | --- | --- |
| Executive Service | Strategic intent and governance-aware workflow direction | Delegating strategic intent to non-executive layers |
| Atlas Institute (Proposed) | Institutional artifact publication, validation, retrieval, and quality reference visibility | Execution orchestration, worker coordination, and approval authority |
| Research Service | Evidence and report generation | Institutional publication governance ownership |
| Memory Service | Historical record persistence | Institutional standards authority |
| Approval Service | Authorization and policy gate decisions | Institutional artifact lifecycle ownership |
| Worker Orchestration | Worker discovery/selection/dispatch/coordination | Institutional policy publication |
| Capability Registry | Capability metadata authority and discovery | Institutional artifact quality governance |
| Control Center | Observational operational visibility | Institutional publication lifecycle management |

## Boundary Rules
- Atlas Institute must preserve existing ownership boundaries.
- Atlas Institute outputs must be traceable and governance-compatible.
- Atlas Institute must not orchestrate runtime execution or approve operational actions.
