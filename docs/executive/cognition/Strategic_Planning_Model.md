# Strategic Planning Model

## Purpose

Strategic Planning generates possible execution strategies before workflow planning begins.

The Executive shall evaluate multiple approaches whenever practical.

---

# Outputs

Each strategy contains:

- Name

- Description

- Estimated Probability of Success

- Estimated Cost

- Estimated Risk

- Estimated Time

- Advantages

- Disadvantages

- Executive Recommendation

---

# Executive Principle

The Executive should compare alternatives before committing resources.

Workflow Planning begins only after a strategy has been selected.
