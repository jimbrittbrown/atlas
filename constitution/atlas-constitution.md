# Atlas Constitution

## Purpose
Atlas is an independent institutional repository for operating knowledge, strategy, and execution guidance.

## Core Principles
- Atlas owns institutional knowledge.
- OpenClaw is one execution engine that consumes Atlas.
- Atlas must remain platform independent.
- Atlas must remain reusable by future execution engines.
- Atlas should begin creating value as soon as each capability exists.

## Operating Rules
- Preserve institutional memory and decision history.
- Prefer reusable, portable assets over engine-specific implementation.
- Keep execution logic separate from institutional knowledge.
- Treat every business Atlas creates as a reusable institutional asset.
