# Documentary Benchmark Library Storage

This directory stores copyright-safe documentary benchmark analysis artifacts.

## Purpose
Store derived observations about documentaries so future analyses can compare structure, pacing, visual rhythm, and narrative effectiveness without reproducing source material.

## Allowed Content
- Benchmark metadata.
- Observation notes.
- Comparative summaries.
- Pattern summaries.
- Review memos.
- Reusable analysis templates.

## Prohibited Content
- Full scripts.
- Transcript dumps.
- Footage.
- Still frames copied as source replacement.
- Any copyrighted content intended to reproduce the original work.

## Structure
```text
research/documentary-benchmark-library/
  README.md
  studies/
    README.md
    benchmark-YYYY-NNNN.md
  observations/
    README.md
    observation-template.md
  templates/
    README.md
    documentary-benchmark-analysis-template.md
```

## Usage Pattern
1. Add one benchmark study per documentary.
2. Keep notes focused on derived observations.
3. Record measurable attributes where available.
4. Link patterns to later analysis only, not to source reproduction.
5. Keep entries append-only.

## Operating Rule
This storage area is for analysis records only and does not affect the production pipeline.
