# Benchmark Studies

Store one study file per documentary benchmark.

## File Naming
Use a stable pattern such as:
- benchmark-YYYY-NNNN.md
- benchmark-YYYY-title-slug.md

## Required Fields
- Benchmark ID
- Documentary title
- Date observed
- Analyst or reviewer
- Source type
- High-level observations
- Pattern notes
- Reuse notes
- Copyright safety note

## Rule
Do not store scripts, transcripts, or footage here.
