# Atlas Repository

This repository is the independent institutional home of Atlas.

## Repository Purpose
Atlas owns institutional knowledge, operating principles, knowledge assets, business-factory resources, and governance records.

OpenClaw remains an execution engine that consumes Atlas rather than owning Atlas content.

## Repository Structure
- Constitution
- Knowledge
- Skills
- Workflows
- Policies
- Decision-Log
- Manifest
- Business Factory

## Current Sprint
Sprint 1 Foundation establishes the initial repository structure, governance artifacts, and starter knowledge content.
